# Brief: Relevance Homepage

## Objective
Merombak "Silo Konten" (tumpukan feed kaku) menjadi "Spiritual Relevance Engine". Homepage harus proaktif merespons kondisi (*state*) rohani pengguna hari ini, menggeser konten paling relevan ke atas, serta merekomendasikan `Hook` khusus.

## Scope
- Penambahan switcher/pilihan `SpiritualState` (*weary, anxious, fresh, grateful, on-fire*).
- Logic reorder feed list di sisi client (Next.js) berdasarkan state tersebut.
- Pengenalan kartu Hook (seperti ajakan minta dukungan doa) saat *state* darurat (e.g. `weary`).
