# Reference Folder

Folder ini menyimpan dokumen referensi yang berguna, tetapi tidak boleh diperlakukan sebagai status operasional utama.

## Isi Folder

- `templates/`
  - template pengisian dokumen
- `ad-hoc-notes/`
  - catatan analisa tambahan atau bundle patch dokumentasi
- `config-snapshots/`
  - snapshot teknis
- `server-config/`
  - referensi konfigurasi server

## Aturan

- jangan gunakan folder ini sebagai penentu status live/runtime
- selalu cross-check ke `09-handover/`, `10-fix-validation-log.md`, dan `07-release-readiness.md`
