# Bug Report: BUG-004

## Title
Registration (Signup) path is completely missing or broken.

## Severity
🛑 **Blocker**

## Priority
Immediate

## Area / Module
Auth / Onboarding

## Environment
Production (`https://www.thechoosentalks.org/register`)

## Preconditions
User attempts to sign up.

## Steps to Reproduce
1. Go to `https://www.thechoosentalks.org/`.
2. Click "Daftar".
3. Redirect goes to `https://www.thechoosentalks.org/login?intent=signup`. 
4. Observe the form (it is the login form, not the signup form).
5. Attempt direct navigation to `https://www.thechoosentalks.org/register`.
6. Observe the result.

## Expected Result
1. `/login?intent=signup` should display Name, Email, Password, and Password Confirmation fields.
2. OR `/register` should be a valid landing page for signup.

## Actual Result
1. `/login?intent=signup` displays only the Login form (Email/Password).
2. `/register` returns a **404 Not Found**.

## Impact
No new users can join the platform. Existing users only.

## Evidence
[Register 404 Snapshot: L15-18]

## Validation Checklist
- [ ] `/register` tidak 404 (Redirect atau render).
- [ ] `/login?intent=signup` menampilkan field pendaftaran lengkap.
