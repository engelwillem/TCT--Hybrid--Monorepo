# Bug Report: BUG-001

## Title
Confusing Primary Login CTA Label ("Buka Blokir").

## Severity
🟠 **Moderate** (UX Friction)

## Priority
Medium

## Suggested Owner: Gemini / Codex

## Area / Module
Auth / UI

## Environment
Production (`https://www.thechoosentalks.org/login`)

## Steps to Reproduce
1. Go to `/login`.
2. Observe the primary action button.

## Expected Result
Label: **"Masuk"** or **"Login"**.

## Actual Result
Label: **"Buka Blokir"** (Unblock).

## Impact
Poor user clarity. Suggests a "locked out" state even for healthy login flow.

## Evidence
Snapshot shows `button "Buka Blokir"`.

## Recommendation
Update translation/label key for the Login screen from `unblock` or similar misleading key to `login`.

## Validation Checklist
- [ ] Button label says "Masuk" or equivalent appropriate text.
