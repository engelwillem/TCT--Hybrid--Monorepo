# Brief: Hook Card System

## Objective
Membangun widget injektor lintas domain. Bukan komponen purna, melainkan "komponen jembatan" (Micro-Content Unit). Widget ini akan ditempatkan pada ujung bawah pembelajaran/bacaan untuk menavigasi user ke modul *Response/Community*.

## Scope
- Komponen agnostik `src/components/cards/HookCard.tsx`.
- Memiliki *props* tipikal: `hookText`, `verseReference`, `relevanceText`, `primaryAction`.
- Variasi visual (Subtle, Highlight, Urgent).
