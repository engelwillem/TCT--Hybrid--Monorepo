# Reporting Policy (DevOps)

Mulai 2026-04-20, semua task dan kegiatan yang terkait DevOps pada repo ini harus membuat laporan di:

`docs/monitoring/DevSecOps Report/`

Format minimum per laporan:
1. tanggal dan scope
2. aktivitas yang dikerjakan
3. file yang diubah
4. hasil/status
5. catatan follow-up
