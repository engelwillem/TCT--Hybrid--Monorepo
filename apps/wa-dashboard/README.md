# WA Dashboard

Web app untuk automasi WA reminder (`wa.thechoosentalks.org`).

## Features (current)
- Login ke backend API
- Session token via cookie
- Route guard untuk halaman private (`/dashboard`, `/reminders`)

## Env
- `NEXT_PUBLIC_WA_API_BASE` (default: `https://api.thechoosentalks.org/api/v1`)

## Commands
- `npm ci`
- `npm run dev`
- `npm run typecheck`
- `npm run build`
